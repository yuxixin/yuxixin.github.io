What is "overfitting"? 什么是“过拟合”？

Jun 20, 2017 in MachineLearning, Concept

This post contains multiple complex mathematic expressions, which you can skip safely.

In machine learning, our purpose is either prediciton or clustering. We focus on an important concept in prediction here – overfitting.

First let’s discuss what is prediction in mathematics. Given a set of training data $x$ and corresponding value $t$, by constructing a model with a set of parameters $S$, predict the value of $t’$ given $x’$.

Put it in a simple way, for example, if we knew the price of estate in Shanghai is 11000/m2 in January, 12000/m2 in February, and 13000/m2 in March. We can easily estimate the price in April is 14000/m2. Although you may not notice the model here is the price $p=m*1000+10000$
